## Design
New `ctxpack/core/code/callgraph.py`:

```python
@dataclass(frozen=True)
class CallEdge:
    caller: str     # qualified-name of the containing function/method
    callee: str     # raw call expression text (e.g. "User", "self._ticks", "logger.info")
    line: int       # 1-indexed line of the call

def build_call_graph(result, file_path) -> list[CallEdge]:
    """For each function/method symbol, walk its body and record every `call` node."""
```

Best-effort. We do NOT resolve `self.method` to `Class.method`, do NOT trace decorator wrappers, do NOT follow imports. CP-019 PageRank still gets useful signal from raw callee names — popular utilities (`logger.info`) get counted, and inside a single class the `Foo.bar`-style attribute calls already match qualified names.
